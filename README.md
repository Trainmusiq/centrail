# Centrail

**TrainMusiq · Vagón 1** — diagnóstico y corrección de afinación de referencia (A440 y afines).

Procesamiento 100% client-side: el audio nunca sale del navegador del usuario. Sin costo operativo, sin servidor.

Estado: en construcción. Especificación completa en [`docs/especificacion.md`](docs/especificacion.md). Prototipo de detección: [`patron440.html`](patron440.html).

Licencia: [GNU GPL v3](LICENSE) (Rubber Band Library, usada para la corrección de pitch, es GPLv2-o-posterior; GPLv3 es compatible y coherente con el espíritu open source del proyecto).

# Centrail (English)

**TrainMusiq · Car 1** — reference pitch diagnosis and correction (A440 and alternate tunings).

100% client-side processing: audio never leaves the user's browser. Zero operating cost, no server.

Status: under construction. Full spec in [`docs/especificacion.md`](docs/especificacion.md) (Spanish). Detection prototype: [`patron440.html`](patron440.html).

License: [GNU GPL v3](LICENSE) (the Rubber Band Library, used for pitch correction, is GPLv2-or-later; GPLv3 is compatible and consistent with the project's open source spirit).
